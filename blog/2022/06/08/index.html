Magnetic pixel image transitions 08/06/2022

Dithering

The first step in the process was to apply dithering to the image so that it could approximate greyscale whilst consisting of only black and white pixels. The dithering algorithm used is called Floyd–Steinberg dithering. This method works by thresholding each pixel and offsetting the error from that threshold onto its neighbours.

Once both the start and finish images have been dithered, we can count the number of black pixels in each, and create a number of particle objects equal to the number of black pixels in the image with the most black pixels.

Javascript implementation

Although this program was initially implemented in the Processing library for Java, a p5.js implementation was also made which allows this process to be performed in the browser.
